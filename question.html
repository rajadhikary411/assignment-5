Question-1: What are the different ways to select an element in the DOM?

To select elements in the DOM, methods like getElementById(), getElementsByClassName(), getElementsByTagName(), querySelector(), and querySelectorAll() are used. These methods allow you to select elements by their ID, class, tag, or CSS selector.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML returns or sets the HTML content inside an element, including HTML tags. innerText returns or sets the visible text content, excluding hidden elements and styles. textContent returns or sets all text, including hidden elements, but without interpreting HTML tags.

Event delegation is a technique in the DOM where a single event listener is attached to a parent element instead of multiple child elements. It leverages event bubbling, allowing events to be handled by the parent even when triggered by child elements. This improves performance and reduces memory usage

Question-4: What is event bubbling in the DOM?

Event bubbling is a mechanism in the DOM where an event starts from the target element and propagates upward through its ancestors in the DOM tree. This allows parent elements to catch events triggered by child elements. It helps in event delegation and managing multiple events more efficiently.

Question-5: How do you create, add, and remove elements using JavaScript?

To create an element: let newElement = document.createElement('div');. To add it: document.body.appendChild(newElement);. To remove it: newElement.remove();.
