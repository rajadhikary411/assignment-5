Question-1: What are the different ways to select an element in the DOM?

To select elements in the DOM, methods like getElementById(), getElementsByClassName(), getElementsByTagName(), querySelector(), and querySelectorAll() are used. These methods allow you to select elements by their ID, class, tag, or CSS selector.

- Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML returns or sets the HTML content inside an element, including HTML tags. innerText returns or sets the visible text content, excluding hidden elements and styles. textContent returns or sets all text, including hidden elements, but without interpreting HTML tags.
